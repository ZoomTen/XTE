XTE - Xtreme Text Editor
Literally Notepad with extras

Overview

XTE is a text editor, forked off the eXtreme Script Editor (XSE).

What's New?

This fork removes the very purpose of the original tool, by gutting out every piece of scripting support. The end result, is that you basically have a glorified Notepad.

And instead of being suited for the GBA Pokémon games, it's now suited for the GB/GBC ones. Specifically, its disassemblies. With fully complete disassembly and mostly documented code, it's become easier than ever to do drastic changes and add many features to the games. As for scripting: just open a text file, hack away and recompile!

Incidentally, you also have these over vanilla XSE. (Maybe I'll make a version of XSE with these changes)

  • Tab indentation is possible in the code editor
  • Adds a binary mode in the calculator
  • Sidebar notes expand to the height of the window
  • Can operate on files originally with Unix/Linux line endings (which is more often these days). Do note that it automatically converts these files to Windows line endings to do so.
  • You can jump to a line number through the command line now, just invoke XTE.exe YourFileHere.asm:14 to jump to line 14 of that file.
