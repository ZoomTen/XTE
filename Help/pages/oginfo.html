XTE - Xtreme Text Editor
Literally Notepad with extras

The History Lesson...

XSE is a program written in 2008 to facilitate easy script editing of the mainline Pokémon games on the Game Boy Advance. XSE complements Advance Map (a map editing tool) quite nicely, and is usually used together.

It was an all-in-one scripting tool, as it can decompile scripts from the game ROM as well as compile scripts to the ROM. Integration with other tools such as the Free Space Finder as well as Advance Trainer comes built-in. It also comes with a hex viewer, ROM resizer, batch compiler, text adjuster... point is: it has it all. It's one of the tools of choice of the GBA hackers of the era.

That being said, there are a few limitations. For example:

  • No syntax highlighting
  • No way to enter TAB inside the code editor
  • Calculator only converts between hexadecimal and decimal
  • Limited space for notes
  • Sidebar not resizeable
  • Written in Visual Basic 6 (okay, maybe it doesn't count)

However, since the tool is extremely(heh) useful, most people didn't mind these limitations.
